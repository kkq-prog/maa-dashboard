# MAA Dashboard (Static) — GitHub Pages

هذا مشروع جاهز للنشر على GitHub Pages.

## الملفات
- index.html
- assets/css/style.css
- assets/js/app.js
- manifest.webmanifest
- sw.js
- assets/icons/icon-192.png
- assets/icons/icon-512.png

## ملاحظة مهمة
هذا الموقع Static (بدون سيرفر/قاعدة بيانات). البيانات تُحفظ محليًا في المتصفح (LocalStorage) على نفس الجهاز فقط.

## نشره على GitHub Pages (سريع)
1) GitHub > New repository
2) ارفع كل الملفات كما هي (لا تغيّر أسماء الملفات)
3) Settings > Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
4) افتح الرابط الذي يعطيك إياه GitHub Pages
