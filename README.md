# موقع HTML جاهز لـ GitHub Pages

هذا المشروع جاهز للنشر كرابط عام باستخدام **GitHub Pages**.

## الملفات
- `index.html` — صفحة الموقع الرئيسية.
- `assets/` — ضع فيها الصور/الملفات (اختياري).
- `.nojekyll` — يخلي GitHub Pages يعرض الملفات كما هي بدون Jekyll.
- `.gitignore` — ملفات يتم تجاهلها (اختياري).

## طريقة رفعه على GitHub
1) افتح GitHub → New repository (Public)
2) ارفع كل ملفات هذا المجلد (أو فك الضغط وارفعها)
3) من الريبو: Settings → Pages
4) Source: Deploy from a branch
5) Branch: `main` و Folder: `/(root)` ثم Save
6) رابطك بيطلع مثل:
   `https://USERNAME.github.io/REPO/`

## تعديل اسم الموقع أو الأيقونة
- افتح `index.html` وعدّل العنوان/المحتوى.
- لو عندك ملفات إضافية (صور…): ضعها داخل `assets/` ثم اربطها داخل HTML.
